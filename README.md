## **📂 Installation & Setup**

### **Run Locally**

1️⃣ **Clone the Repository**

```bash
git clone https://github.com/mohsinhamza7248/Portfolio.git
```

2️⃣ **Navigate to the Project Folder**

```bash
cd your-portfolio
```

3️⃣ **Install Dependencies**

```bash
npm install
```

4️⃣ **Run the Development Server**

```bash
npm run dev
```

5️⃣ **Open in Browser**

```
http://localhost:3000
```
